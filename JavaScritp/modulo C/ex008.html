Eventos Dom

Sem a Dom teriamos muito dificuldade de acesso ao elementos da nossa arvore de navegação, a DOM é um facilitador que permite realizar dentro do navegador, do html5 moidificações nos elementos atraves do java script.

Cada projeto tem a sua arvore, não precisa decorar, apenas entende como ela funciona

Tecnica para acessar os elementos: TagName ; ID ; Name; ClassName; querySelector

No exemplo abaixo usaremos o evento do mouse; mas existe diversos eventos que podem ser aplicados:consultar o site: event reference do MDN

OS eventos são coisas que eu posso fazer com os elementos

Para disparar um evento / tratar um evento precisamos utilizar o comando Funções

A função ou funcionalidade é um conjunto de linha( bloco {} = linhas), de codigos que serão execultada somente quando o evento acontecer

formato do comando function = function ação(param){},

Posso aplicar o comando direto no html id="area" onclick="clicar()" onmouseenter="entra()" onmouseout="sair()">. div -> Interaja... / mas isso fara com que fique poluido

Então podemos aplicar o comando pelo proprio java scritpt utilizando os seguites comandos:

var a = window.document.getElementById('area') a.addEventListener('click', clicar) a.addEventListener('mouseenter',entra) a.addEventListener('mouseout',sair) / que tera a função de ouvir os comando acontecerem e execultar as açoes solicitadas
