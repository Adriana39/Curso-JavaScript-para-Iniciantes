True ou False

O resultado sempre será boleano, ou seja verdadeiro ou falso.
Var a = 8 // var b = 10
a>8 -> false
b>a -> true
a>10*2 -> false
b <2-1 -> false
b>2*3 -> true
var preço = 30.000 -> preço>=200.50 -> false -> preço<=200.50 true
var idade = 18 -> idade <18 false> idade>=18
var curso = 'java script' - > curso=='java script' -> true -> curso== 'video' false
var n1 = 5 // var n2 = 5
n1=!n2 -> false

Identidade ou Identico

== iguais ( mesmo valor e tipos diferentes) e === identicos ( mesmo valor e mesmo tipo)

5=='5' -> true
5==5 -> true
5==='5' -> false
5===5 -> true

Operadores Logicos

! (exclamação) Negação / não
negação - > operando unário, retorna os valores true ou false // uma coisa que não é verdadeira é falsa // outra coisa que nao é falsa é verdadeira

&& ( 2 e comercial) conjução / e
esse operador binario, temos 2 valores // os valores precisa ser true, diferente disso o resultado é falso

|| (2 barra ) disjunção / ou
esse operador é binario, tem 2 valores logico, basta ter um verdadeiro para que o resultado seja verdadeiro, apenas falso quando os dois valores são falsos
